## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express, MongoDB

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd client
```

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
